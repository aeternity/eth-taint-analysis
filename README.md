## eth-tracker
Tracking the ether flow on block chain.

## Nodejs RESTful API

clone the project

### Installations

    npm install

### Run

    npm run start

### API

    http://localhost:3000/track/<address>/<depth>
    http://localhost:3000/track/0xb3764761e297d6f121e79c32a65829cd1ddb4d32/3

### Web Interface

    track/index.html
